# 人生K线图

这是一个基于生辰八字分析的Web应用程序。用户可以输入自己的生辰信息，系统会调用大模型API进行命理解析，并展示分析结果。

## 功能特点

- 用户友好的生辰信息输入界面
- 八字命理分析展示
- 模块化设计，易于扩展
- 抽象接口设计，支持替换不同的大模型提供商

## 开发说明

项目采用Flask框架构建，实现了前后端分离的架构。目前使用模拟数据进行展示，后续可以集成真实的通义千问API或其他大模型服务。

### 核心组件

- `app/core/interfaces.py`: 定义了核心接口
- `app/core/mock_model_provider.py`: 模拟大模型提供者
- `app/core/fortune_analyzer.py`: 命运分析器
- `app/models/birth_chart.py`: 生辰八字数据模型
- `app/routes/api.py`: RESTful API接口
- `app/routes/web.py`: Web页面路由
